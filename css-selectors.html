Selector Syntax Description Explanation
a, b Multiple element selector I want styling to apply to both of these things, the descendant
a b Descendant selector I want to select everything contained within something else
a > b Child selector immediate child for immediate children but not grandchildren or great grandchildren of an element
a+b Adjacent Sibling selector something that comes immediately after something else-- so two elements that are sisters to each other in the tree rather than one being the parent or containing the other
[a=b] Attribute selector attribute selector that let us say that inputs that are numbers should be treated as different from text inputs
a:b Pseudoclass selector things like when I'm hovering over an element,
a::b Pseudoelement selector before the contents of a page or after a contents of the page
